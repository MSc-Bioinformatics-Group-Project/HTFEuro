All about SNPEuro

SNPEuro (Single Nucleotide Polymorphism Euro)is a web application that aims at retrieving SNP information for a genomic region of interest, and allows users to download Results and statistical analysis in a format of tsv file.

Users are able to select either a genomic coordinate, SNP name or a gene of interest. The name, genomic position, genotype frequencies and allele frequencies for each SNP will be retrieved.

This webapp was developed as part of the Queen Mary University of London (QMUL) Bioinformatics masters degree program.

Developers of this app by Vi Bui, Uppy Gill, Liam Johnson, Neha Khairnar. The code can be found on our Github
